## Angular/Akita
状態管理ライブラリ。  
NgRXより簡単で使いやすい。  

### セットアップ
npm、yarnでインストールする方法もあるが、Angularのcliで追加した方がコードの補完もしてくれて楽。  
```
ng add @datorama/akita
```
